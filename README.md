# 🌐 **Facebook**

**A lightweight social media platform for learning full-stack development**

![License: MIT](https://img.shields.io/badge/License-MIT-green) ![Node.js](https://img.shields.io/badge/Node.js-14%2B-blue) ![MongoDB](https://img.shields.io/badge/MongoDB-red)

---

## ✨ **What is this?**

Clean educational project to learn how social networks work. No fake data, no bloat, just real working features.

**Works:** Register, Login, Create Posts, Upload Images/Videos, Edit/Delete Posts, Feed, Video Player

**Coming Soon UI:** Comments, Likes, Share, Notifications, Chat (buttons show "Coming Soon" modal)

---

## ⚡ **Quick Start**

```bash
# 1. Clone & install
git clone (URL)
cd facebook
npm install

# 2. Run server
node server.js

# 3. Open browser (Backend) )
http://localhost:5000
```

---

## 🔌 **API Endpoints**

| Method | Endpoint | Description |
|--------|----------|-------------|
| POST | `/api/auth/register` | Sign up |
| POST | `/api/auth/login` | Sign in |
| POST | `/api/post/upload` | Upload post (file + title + caption) |
| GET | `/api/post/all?page=1` | Get all posts |
| PUT | `/api/post/:id` | Edit post |
| DELETE | `/api/post/:id` | Delete post |

---

## 🛠️ **Tech Stack**

- **Frontend:** HTML + CSS + Vanilla JavaScript (no frameworks)
- **Backend:** Node.js + Express.js
- **Database:** MongoDB + Mongoose
- **Auth:** JWT + Bcrypt
- **Uploads:** Multer
